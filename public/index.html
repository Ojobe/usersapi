## Getting Started
These instructions will guide you through running this app
### Prerequisites

What things you need to run this App

  • Internet connection(required)
  • Web browser (preferably Google Chrome)
  • POSTMAN application or Any Rest Client , preferably POSTMAN
  • JSONview a google chrome extension to help format the views (optional)
#### Routes
  • /users
  • /users/:userId
  • /users/firstname/:firstname
  • /users/lastname/:lastname
  • /users/gender/:gender
  • /users/sort/firstname
  • /users/sort/lastname

##### Operations on POSTMAN

Launch the application by clicking here

  1. To view all users

    Type this url https://radiant-springs-82648.herokuapp.com/users in the browser or in POSTMAN Address bar and click on send

    Note the HTTP Method must be GET

  2. To add a user

    Type this url https://radiant-springs-82648.herokuapp.com/users in POSTMAN Address bar ,
    below the url in POSTMAN app select Body
    below the Body select x-www-form-urlencoded
    then enter the fieldnames in the KEY and the values in the VALUE

    Note the http method must be POST

  3. To view a user by id

    Type this url https://radiant-springs-82648.herokuapp.com/users/id in the browser or in POSTMAN Address bar and click on send, where the id is any of the user id

    e.g https://radiant-springs-82648.herokuapp.com/users/5d34badd9ea3370004337c71

    Note the http method must be GET

  4. To view user by firstname

    Type this url https://radiant-springs-82648.herokuapp.com/users/firstname in the browser or in POSTMAN Address bar and click on send, where the firstname is any of the user firstname

    e.g https://radiant-springs-82648.herokuapp.com/users/chinedu

    Note the http method must be GET

  5. To view user by lastname

    Type this url https://radiant-springs-82648.herokuapp.com/users/lastname in the browser or in POSTMAN Address bar and click on send, where the lastname is any of the user lastname

    e.g https://radiant-springs-82648.herokuapp.com/users/ben

    Note the http method must be GET

  6. To view user by gender

    Type this url https://radiant-springs-82648.herokuapp.com/users/gender in the browser or in POSTMAN Address bar and click on send, where the gender is either male or female

    e.g https://radiant-springs-82648.herokuapp.com/users/male

    Note the http method must be GET

  7. To update some fields say firstname, date of birth of a user with a given id

    Type this url https://radiant-springs-82648.herokuapp.com/users/id in POSTMAN Address bar ,
    below the url in POSTMAN app select Body
    below the Body select x-www-form-urlencoded
    then enter the field name that you want to update in the KEY and the value you want it to be changed to in the VALUE
    where the id is any of the users id

    e.g https://radiant-springs-82648.herokuapp.com/users/5d34badd9ea3370004337c71

    Note the http method must be PATCH

  8. To update all fields of a user with a given id

    Type this url https://radiant-springs-82648.herokuapp.com/users/id in POSTMAN Address bar ,
    below the url in POSTMAN app selectBody
    below the Body select x-www-form-urlencoded
    then enter the all the fields in the KEY and the value you want them to be changed to in the VALUE then click send
    where the id is any of the users id

    e.g https://radiant-springs-82648.herokuapp.com/users/5d34badd9ea3370004337c71

    Note the http method must be PUT

  9. To delete a user with a given id

    Type this url https://radiant-springs-82648.herokuapp.com/users/id in POSTMAN Address bar , then click send
    where the id is any of the users id

    e.g https://radiant-springs-82648.herokuapp.com/users/5d34badd9ea3370004337c71

    note the http method must be DELETE

  10. To sort the users by firstname

    Type this url https://radiant-springs-82648.herokuapp.com/users/sort/firstname in POSTMAN Address bar , then click send

    e.g https://radiant-springs-82648.herokuapp.com/users/sort/firstname

    Note the http method must be GET

  11. To sort the users by lastname

    Type this url https://radiant-springs-82648.herokuapp.com/users/sort/lasttname in POSTMAN Address bar , then click send

    e.g https://radiant-springs-82648.herokuapp.com/users/sort/lastname

    Note the http method must be GET
